Fecha: 2026-04-03
Problema: Error al analizar imagen con IA

═══════════════════════════════════════════════════════════════════════════════
  PROBLEMA DETECTADO
═══════════════════════════════════════════════════════════════════════════════

Al usar "Analizar imagen con IA" aparecía:

  ❌ Error: Error en formato de la IA: Unexpected end of JSON input
  
  Verifica que el API Key sea correcto y la imagen sea clara.

═══════════════════════════════════════════════════════════════════════════════
  CAUSA
═══════════════════════════════════════════════════════════════════════════════

La API de Google Gemini devolvía:
  • Respuesta JSON incompleta (truncada)
  • Respuesta vacía (API Key inválido)
  • JSON mal formado

═══════════════════════════════════════════════════════════════════════════════
  SOLUCIÓN APLICADA
═══════════════════════════════════════════════════════════════════════════════

✅ Validación de respuesta JSON
   → Detecta si la respuesta es inválida antes de procesarla

✅ Validación de contenido
   → Verifica que la IA devolvió algo útil

✅ Reparación automática de JSON truncado
   → Intenta reparar JSON incompleto agregando llaves/corchetes

✅ Mensajes de error más claros
   → Indica exactamente qué salió mal y cómo solucionarlo

═══════════════════════════════════════════════════════════════════════════════
  CÓMO PROBAR
═══════════════════════════════════════════════════════════════════════════════

1. Recarga tu aplicación (Ctrl+Shift+R)

2. Ve a "Importar cifrado desde imagen"

3. Ingresa tu API Key de Google Gemini
   (Si no tienes: https://aistudio.google.com/app/apikey)

4. Selecciona una imagen clara con acordes

5. Haz clic en "🤖 Analizar imagen con IA"

6. Espera el resultado

RESULTADO ESPERADO:
  ✅ Si funciona: Verás los acordes detectados
  ✅ Si falla: Verás un mensaje de error más claro y útil

═══════════════════════════════════════════════════════════════════════════════
  SI SIGUE FALLANDO
═══════════════════════════════════════════════════════════════════════════════

1. Verifica tu API Key:
   → https://aistudio.google.com/app/apikey
   → Asegúrate de tener cuota disponible

2. Prueba con una imagen más simple:
   → Divide canciones largas en secciones
   → Usa imágenes más claras

3. Revisa Console (F12):
   → Busca mensajes de error detallados
   → Copia el error completo

4. Lee la documentación:
   → FIX_ERROR_IA_JSON.md
