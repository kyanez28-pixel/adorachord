═══════════════════════════════════════════════════════════════════════════════
  PROBLEMAS COMUNES Y SOLUCIONES
═══════════════════════════════════════════════════════════════════════════════

PROBLEMA: "syncStart is not defined"
SOLUCIÓN: 
  → Verifica que estás usando index-FIXED.html
  → Busca en el HTML: "function syncStart"
  → Si no existe, copia FIX_GUARDAR_ACORDES.js antes de </body>

PROBLEMA: Service Worker no se registra
SOLUCIÓN:
  → Verifica que sw.js existe en la raíz
  → DevTools → Application → Service Workers → Unregister
  → Recarga con Ctrl+Shift+R

PROBLEMA: "Error al guardar" persiste
SOLUCIÓN:
  → Abre Console y copia el error exacto
  → Verifica typeof syncStart (debe ser "function")
  → Consulta ANALISIS_ERROR_GUARDAR.md

PROBLEMA: Caché antigua (1.1.1) no se elimina
SOLUCIÓN:
  → DevTools → Application → Cache Storage
  → Haz clic derecho en caché antigua → Delete
  → Recarga la página

PROBLEMA: App no funciona offline
SOLUCIÓN:
  → Verifica que Service Worker está activo
  → Verifica que la caché 1.2.0 existe
  → Recarga con Ctrl+Shift+R
  → Intenta offline de nuevo
